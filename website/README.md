# Curl Bot 官方網站

這是 Curl Bot 的官方網站，採用現代化的深色主題設計，與機器人控制台保持一致的視覺風格。

## 📁 檔案結構

```
website/
├── index.html          # 主頁
├── docs.html           # 指令文檔頁面
├── dashboard.html      # 控制台登入頁面
└── static/
    ├── css/
    │   └── style.css   # 統一樣式表
    └── js/
        └── main.js     # 互動腳本
```

## 🎨 設計特色

- **深色主題**: 與機器人控制台一致的配色方案
  - 背景色: `#1a1a1a`
  - 卡片背景: `#2b2b2b`
  - 主色調: `#2563eb` (藍色)

- **響應式設計**: 完美支援手機、平板、桌面等各種裝置

- **流暢動畫**: 包含淡入、滑動、浮動等多種動畫效果

- **現代化 UI**: 採用卡片式設計、漸層效果、陰影等現代元素

## 📄 頁面說明

### index.html - 主頁
包含以下區塊：
- 導航欄

### dashboard.html - 控制台
- 控制台登入引導頁

- 資訊卡片展示

## 🚀 使用方式

### 本地預覽

1. 直接在瀏覽器中打開 `index.html` 即可預覽

2. 或使用簡易的本地伺服器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```
   然後訪問 `http://localhost:8000`

### 整合到現有專案

1. 將 `website` 資料夾放在專案根目錄

2. 在網頁伺服器中設定靜態檔案服務：
   ```python
   # 範例：在 server.py 中添加
   @app.route('/website')
   def website():
       return send_from_directory('website', 'index.html')
   ```

3. 訪問 `http://your-domain/website` 查看網站

## 🎯 功能亮點

1. **平滑滾動**: 點擊導航連結平滑滾動到對應區塊

2. **響應式導航**: 移動端自動切換為漢堡選單

3. **動態數字**: 統計數字會在進入視窗時動畫計數

4. **懸停效果**: 所有互動元素都有流暢的懸停效果

5. **自動高亮**: 滾動時自動高亮當前區塊的導航項

## 🔧 客製化

### 修改顏色
在 `static/css/style.css` 的 `:root` 區塊修改 CSS 變數：

```css
:root {
    --primary-color: #2563eb;     /* 主色調 */
    --background: #1a1a1a;        /* 背景色 */
    --surface: #2b2b2b;           /* 卡片背景 */
    /* ... 更多變數 */
}
```

### 修改內容
直接編輯對應的 HTML 檔案即可

### 添加新頁面
1. 複製現有的 HTML 檔案作為模板
2. 修改內容
3. 在導航欄中添加連結
